# Design Direction

- Theme: dusk command bunker overlooking a heated border crossing.
- Palette: slate navy, oxidized teal, ember orange, pale sand.
- Shape language: framed panels, glowing build pads, lane heat maps, restrained military-futurist UI.
- Motion: wave pulses, turret scan sweeps, projectile streaks, status chips that light up on state changes.
- Twist expression: mod cores feel like physical cartridges inserted into turret housings.
